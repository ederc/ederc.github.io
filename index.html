------------------   Software   ------------------

A Julia wrapper for gb GB.jl Library implemented by Christian Eder. Interface between Singular and gb which also provides a benchmark suite, part of the OSCAR computer algebra system.
A C library for Gröbner basis computations gb Library implemented by Christian Eder. Implementation of Faugère's F4 algorithm using GBLA for the linear algebra part.
A C library for specialized linear algebra for Gröbner basis computations GBLA Library implemented by Brice Boyer, Christian Eder, Jean-Charles Faugère and Fayssal Martani. Improved handling of data structures, improved parallelization on multicore CPUs and NUMAs.The library also comes with a huge database for example matrices.
A library for exact linear algebra LELA Library implemented by Bradford Hovinen and Fayssal Martani. Supported for structured hybrid sparse-dense matrices. Further testing and improving this library.
A library for computing Gröbner bases MathicGB Library written by Bjarke Hammersholt Roune and Mike Stillman. Supported Bjarke by implementing linear algebra for Gröbner basis computations.
A library for testing different parallel scheduler for dense linear algebra pLA Library written to test various parallel schedulers (like OpenMP, Intel TBB, xKAAPI, etc.) and compare their behaviours for dense linear algebra, esp. Gaussian Elimination.
A computer algebra system Singular Implemented fast signature-based Gröbner basis algorithms covering many variants like F5, G2V, GVW, etc.
Benchmark file in Singular language singular-benchmarks Various Gröbner basis benchmarks for the Singular computer algebra system.
A library for special-purpose memory allocation xmalloc Implemented a special-purpose memory allocation for the computer algebra system Singular (used Singular's omalloc implementation as a basis).
